clamp_ :: Dynamic -> HsReal -> HsReal -> IO () #

Clamp all elements, inplace, in the tensor into the range [min_value, max_value]. ie:

        { min_value, if x_i < min_value
  y_i = { x_i,       if min_value ≤ x_i ≤ max_value
        { max_value, if x_i > max_value
